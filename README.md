# 关系图谱功能开发数据结构分析与实践


### 简介

根据搜索关键字，找出该关键字相关联的数据信息并生成关系图谱。目标源数据为三元组结构，即实体、关系、属性。
项目前端使用Vue，后端使用SpringMVC+MyBatis，数据库使用MySQL。
